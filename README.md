# Sobre o projeto

O site Camp.in foi desenvolvido como um desafio pessoal para colocar em prática os conhecimentos adquiridos durante o programa Explorer da Rocketseat, 
utilizando HTML e CSS, foram criadas tanto a versão mobile quanto a desktop, aplicando conceitos de responsividade e mobile first.

## Layout web
O site pode ser visualizado no seguinte endereço: 
https://mcarnelos.github.io/projeto-camp.in/
